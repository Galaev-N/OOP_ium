# ProductCatalog - Управление коллекцией товаров

##  Описание проекта

`ProductCatalog` - это класс для управления коллекцией товаров (`Product`). Он предоставляет удобный интерфейс для добавления, удаления, поиска, сортировки и фильтрации товаров с поддержкой индексации и итерации.


## Возможности

- Добавление и удаление товаров
- Поиск товара по ID
- Доступ к элементам по индексу (`catalog[i]`)
- Итерация по каталогу (`for product in catalog`)
- Сортировка по цене
- Фильтрация товаров в наличии
- Удаление по индексу
- Валидация типов и обработка ошибок
- Создание каталога из списка товаров

## Методы класса

| Метод | Описание | Возвращаемое значение |
|-------|----------|----------------------|
| `A(obj)` | Добавляет товар в каталог | `str` - сообщение об успехе |
| `R(obj)` | Удаляет товар из каталога | `str` - сообщение об успехе |
| `GAll()` | Возвращает все товары | `list[Product]` |
| `FBI(id)` | Ищет товар по ID | `Product` или `None` |
| `RAI(index)` | Удаляет товар по индексу | `Product` - удаленный товар |
| `SBP()` | Сортирует каталог по цене | `list[Product]` |
| `GAva()` | Возвращает товары в наличии | `list[Product]` |
| `__len__()` | Возвращает количество товаров | `int` |
| `__getitem__(index)` | Доступ по индексу | `Product` |
| `__iter__()` | Итератор | `iterator` |

1. Создание каталога и добавление товаров

```python
from collection import ProductCatalog
from model import Product

# Создаем каталог
catalog = ProductCatalog()

# Добавляем товары
catalog.A(laptop)
catalog.A(mouse)
catalog.A(keyboard)

print(len(catalog))
```  
2. Доступ по индексу и итерация

```python
# Прямой доступ по индексу
first_product = catalog[0]
last_product = catalog[-1]

# Итерация
for product in catalog:
    print(product.name)
```
3. Сортировка по цене

```python
# Сортировка по возрастанию цены
catalog.SBP()

# Теперь товары отсортированы от дешевых к дорогим
cheapest = catalog[0]
most_expensive = catalog[-1]
```
4. Фильтрация товаров в наличии

```python
# Получаем только те товары, у которых quantity > 0
available = catalog.GAva()

for product in available:
    print(f"{product.name} - {product.quantity} шт.")
```
5. Поиск по ID

```python
# Поиск товара по уникальному идентификатору
product = catalog.FBI("#A12345")
if product:
    print(f"Найден: {product.name}")
```
6. Удаление товаров

```python
# Удаление по значению
catalog.R(mouse)

# Удаление по индексу
removed = catalog.RAI(0)
print(f"Удален: {removed.name}")
```

⚠️ Обработка ошибок

Класс корректно обрабатывает следующие ошибочные ситуации:

Ошибка	Ситуация
TypeError	Попытка добавить объект, не являющийся Product
TypeError	Неверный формат каталога при создании
ValueError	Удаление товара, которого нет в каталоге
ValueError	Создание каталога с невалидным списком
IndexError	Обращение или удаление по неверному индексу
