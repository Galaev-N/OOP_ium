# Иерархия классов товаров и расширенный каталог

## Описание проекта

В рамках лабораторной работы реализована иерархия классов на основе базового класса `Product`. Созданы классы-наследники для различных категорий товаров: `Food` (продукты питания), `Techic` (техника) и `Estate` (недвижимость). Класс `ProductCatalog` расширен для работы с полиморфной коллекцией, поддерживающей объекты всех типов-наследников.

## Архитектура классов


### Базовый класс `Product`
- Содержит общие атрибуты: `name`, `price`, `quantity`, `producer`, `cost_price`, `_id`, `description`, `comments`, `mark`
- Реализована валидация всех полей через модуль `validate.py`
- Методы: `__str__`, `__repr__`, `__eq__`, `feedback()`, свойства `mark` и `info`

### Класс `Food` (Продукты питания)
**Дополнительные атрибуты:**
- `calories` - калорийность (ккал на 100г)
- `expiration_per` - срок годности (в днях)

**Специфические методы:**
- `fits(our_day)` - рассчитывает дату, до которой продукт годен к употреблению

### Класс `Techic` (Техника)
**Дополнительные атрибуты:**
- `warranty_months` - гарантийный срок (месяцы)
- `power_consumption` - потребляемая мощность (Вт)

**Специфические методы:**
- `check()` - случайная "проверка" выгодности предложения

### Класс `Estate` (Недвижимость)
**Дополнительные атрибуты:**
- `location` - координаты (кортеж из двух чисел)
- `proportions` - площадь (кв. м)
- `free` - статус (1 - свободно, 0 - занято)

**Специфические методы:**
- `how_long(yo_loc)` - рассчитывает расстояние до указанной точки

## Класс ProductCatalog (расширенная версия)

Класс для управления полиморфной коллекцией товаров. Поддерживает объекты `Product`, `Food`, `Techic`, `Estate`.

### Методы класса

| Метод | Описание | Возвращаемое значение |
|-------|----------|----------------------|
| `A(obj)` | Добавляет товар в каталог | `str` - сообщение об успехе |
| `R(obj)` | Удаляет товар из каталога | `str` - сообщение об успехе |
| `GAll()` | Возвращает все товары | `list` |
| `FBI(id)` | Ищет товар по ID | `Product` или `None` |
| `RAI(index)` | Удаляет товар по индексу | `Product` - удаленный товар |
| `SBP()` | Сортирует каталог по цене | `list[Product]` |
| `GAva()` | Возвращает товары в наличии/свободные | `ProductCatalog` |
| `GD()` | Фильтрация - только техника | `ProductCatalog` |
| `GF()` | Фильтрация - только продукты питания | `ProductCatalog` |
| `GE()` | Фильтрация - только недвижимость | `ProductCatalog` |
| `__len__()` | Возвращает количество товаров | `int` |
| `__getitem__(index)` | Доступ по индексу | `Product` |
| `__iter__()` | Итератор | `iterator` |
